# 🎓 Student Placement Analysis Dashboard (Tableau)

## 📌 Project Overview

The **Student Placement Analysis Dashboard**is an interactive Tableau project designed to analyze placement trends, identify student skill gaps, and provide salary and career insights. The dashboard enables users to explore placement performance across departments, technical skills, academic scores, and salary packages through dynamic visualizations.

---

## 🎯 Objectives

- Analyze overall student placement performance.
- Identify skill gaps affecting placements.
- Compare placement trends across departments.
- Analyze salary packages and career insights.
- Support data-driven decision making for training and placement teams.

---

## 🛠️ Tools & Technologies

- **Tableau Desktop**
- Data Visualization
- Dashboard Design
- Interactive Filters
- Calculated Fields
- KPIs
- Charts & Graphs

---

## 📊 Dashboard Features

### 1. Placement Analysis Dashboard

Provides an overview of placement statistics, including:

- Total Students
- Placed Students
- Not Placed Students
- Average Salary Package
- Maximum Salary Package
- Department-wise Placement Analysis
- Gender-wise Placement Distribution
- Placement vs CGPA Analysis

---

### 2. Skill Gap Analysis Dashboard

Analyzes the relationship between technical skills and placement outcomes.

Includes:

- Placement vs Tableau Skill
- Placement vs Python Skill
- Placement vs SQL Skill
- Average Skill Scores
- Branch-wise Skill Comparison

This dashboard helps identify the technical skills that contribute most to successful placements.

---

### 3. Salary & Career Insight Dashboard

Provides insights into salary trends and career readiness.

Features include:

- Top 10 Students by Salary Package
- Branch-wise Salary Comparison
- Placement Status vs Resume Score
- Backlog Analysis
- Career Insight Metrics

---

## 📈 Key Insights

- Students with higher Python, SQL, and Tableau skills have better placement rates.
- Resume quality significantly impacts placement success.
- Some departments consistently achieve higher placement percentages.
- Students with fewer backlogs generally receive better placement opportunities.
- Salary packages vary across departments and academic performance.
